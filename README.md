# dotfiles

Dotfiles for my archlinux setup using i3-gaps WM.

## What's Included?

Config files for:

- i3-gaps
- compton
- polybar
- xinitrc
- zshrc

## Fonts

- Fantasque Sans

```console
foo@bar:~$ sudo pacman -S ttf-fantasque-sans-mono
foo
```

- Iosevka

```console
foo@bar:~$ sudo yay -S ttf-iosevka
foo
```
